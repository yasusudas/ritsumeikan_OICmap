# 立命館キャンパスマップ

ブラウザで動作するフロア切替付きの地図アプリです。表示は `フロア/` 内の SVG を使い、教室検索は `pdf/` 内の OCR 埋め込み PDF を参照します。

## 起動方法

Finder から簡単に起動する場合:

`start-map.command` をダブルクリックしてください。

ターミナルから起動する場合:

```bash
npm install
npm run dev
```

開発サーバー起動後、ブラウザで `http://127.0.0.1:5173/` を開いてください。

このアプリは `index.html` を `file:///...` で直接開いても動作しません。必ずローカルサーバー経由で開いてください。

ポートを固定したい場合:

```bash
npm run dev -- --host 127.0.0.1 --port 4173
```

## できること

- 上部タブで `1F` から `5F` の地図を切り替え
- ピンチイン・ピンチアウトで拡大縮小
- 1本指ドラッグで地図移動
- 右下の `+` / `-` ボタンでズーム操作
- マウスホイールでズーム、ドラッグで移動
- 教室名を 1F〜5F 横断で検索し、該当位置をハイライト

## 本番ビルド

```bash
npm run build
```
