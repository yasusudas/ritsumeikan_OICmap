# 立命館キャンパスマップ

ブラウザで動作するフロア切替付きの地図アプリです。表示は `floor_img/` 内の SVG を使い、教室検索は `public/manual-search-index.json` の座標データを参照します。

## 起動方法

Finder から簡単に起動する場合:

`start-map.command` をダブルクリックしてください。

ターミナルから起動する場合:

```bash
npm install
npm run dev
```

開発サーバー起動後、ブラウザで次のどちらかを開いてください。

- 閲覧用サイト: `http://127.0.0.1:5173/`
- 編集用サイト: `http://127.0.0.1:5173/editor/`

このアプリは `index.html` を `file:///...` で直接開いても動作しません。必ずローカルサーバー経由で開いてください。

ポートを固定したい場合:

```bash
npm run dev -- --host 127.0.0.1 --port 4173
```

## できること

- 上部タブで `1F` から `5F` の地図を切り替え
- ピンチイン・ピンチアウトで拡大縮小
- 1本指ドラッグで地図移動
- 右下の `+` / `-` ボタンでズーム操作
- マウスホイールでズーム、ドラッグで移動
- 教室名を 1F〜5F 横断で検索し、該当位置をハイライト
- 編集用サイトでは SVG をクリックして検索用テキスト位置を記録し、`manual-search-index.json` を書き出し

## 本番ビルド

```bash
npm run build
```
