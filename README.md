# 立命館キャンパスマップ

ブラウザで動作するフロア切替付きのPDF地図アプリです。

## 起動方法

Finder から簡単に起動する場合:

`start-map.command` をダブルクリックしてください。

ターミナルから起動する場合:

```bash
npm install
npm run dev
```

開発サーバー起動後、ブラウザで `http://127.0.0.1:5173/` を開いてください。

このアプリは `index.html` を `file:///...` で直接開いても動作しません。必ずローカルサーバー経由で開いてください。

ポートを固定したい場合:

```bash
npm run dev -- --host 127.0.0.1 --port 4173
```

## できること

- 上部タブで `1F` から `5F` のPDFを切り替え
- ピンチイン・ピンチアウトで拡大縮小
- 1本指ドラッグで地図移動
- 右下の `+` / `-` ボタンでズーム操作
- マウスホイールでズーム、ドラッグで移動

## 本番ビルド

```bash
npm run build
```
