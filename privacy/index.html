プライバシーポリシー

立命館OICマップにおける個人情報・データの取り扱い方針です。

最終更新日：2026年5月11日

1. 運営者について

立命館OICマップ（以下「本サービス」）は、個人が開発・運営する非公式サービスです。立命館大学および関連機関とは一切関係ありません。

2. 取得する情報

本サービスでは、サービス改善と安定運用のため、以下の情報を取得することがあります。

  • ページ閲覧情報、参照元、端末種別、ブラウザ、OS、地域などのアクセス解析情報
  • お問い合わせフォームに入力された内容
  • ブラウザに保存される言語設定や編集用の下書きデータ

3. 利用目的

取得した情報は、以下の目的で利用します。

  • 本サービスの表示、検索、編集機能を提供するため
  • 利用状況を把握し、機能改善や不具合調査を行うため
  • お問い合わせへの返信や必要な連絡を行うため
  • 不正利用や過度なアクセスを防止するため

4. アクセス解析について

本サービスでは、Vercel Web Analytics を利用しています。Vercel Web Analytics は、ページビューや利用環境などの集計情報を取得します。Vercel の公式ドキュメントによると、Web Analytics は個人やIPアドレスに紐づけられない匿名のデータポイントとして記録され、第三者Cookieは使用しません。

Vercel Web Analytics のプライバシーに関する説明： https://vercel.com/docs/analytics/privacy-policy

5. Cookie・ローカルストレージについて

本サービスは、言語設定や編集用サイトの下書き保存などのために、ブラウザのローカルストレージを使用します。これらのデータは利用者のブラウザ内に保存され、利用者自身で削除できます。

また、Google Forms など外部サービスを開いた場合、その外部サービスがCookie等を使用することがあります。外部サービスでの情報管理は、それぞれのサービスのポリシーをご確認ください。

6. 第三者提供について

法令に基づく場合を除き、取得した情報を本人の同意なく第三者へ提供することはありません。ただし、アクセス解析、ホスティング、お問い合わせフォームなど、本サービスの運営に必要な範囲で外部サービスに情報が送信される場合があります。

7. 外部リンクについて

本サービスには、大学資料、Google Forms、その他外部サイトへのリンクが含まれる場合があります。外部サイトでの個人情報の取り扱いについて、本サービスの運営者は責任を負いません。

8. プライバシーポリシーの変更

本ポリシーは予告なく変更することがあります。変更後のポリシーは、このページに掲載した時点で効力を生じます。

9. お問い合わせ

本ポリシーに関するお問い合わせは、トップページの「お問い合わせ」からご連絡ください。
